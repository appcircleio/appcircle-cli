# Appcircle CLI Migration Guide from Version 1.x to 2.x

## Introduction
This guide provides instructions for migrating Appcircle CLI commands from version 1.x to version 2.x. It outlines the changes in the command structure and provides examples for updating your commands accordingly.


## Update Command Syntax
- Modify the syntax of commands according to the changes in version 2.x. Use the following format:
  - Old command syntax:
    ```bash
    appcircle oldCommand <subcommand> [options] [arguments]
    ```
  - New command syntax:
    ```bash
    appcircle <command> <subcommand> ... <subcommand> [options] [arguments]
    ```
