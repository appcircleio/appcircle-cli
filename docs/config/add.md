# `appcircle config add`

Add a new Appcircle CLI configuration environment.

```plaintext
appcircle config add <value>
```            

## Arguments

| Argument                                                                                                                                 | Description                                             |
| --------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------- |
| `value`                                                                                                        | New configuration environment name
