# `appcircle organization user invite`

Invite user to organization.

```plaintext
appcircle organization user invite [options]
```

## Options

```plaintext
      --organizationId <uuid>  Organization ID [Optional] (default: "current")
      --email <string>         Email
      --role <string>          Role

```
