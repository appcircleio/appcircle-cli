# `appcircle organization role view`

View roles of the given userId within the organizationId.

```plaintext
appcircle organization role view [options]
```

## Options

```plaintext
      --organizationId <uuid>  Organization ID [Optional] (default: "current")
      
      --userId <uuid>          User ID
```
