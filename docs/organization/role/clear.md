# `appcircle organization role clear`

Remove all roles from the given userId within the organizationId.

```plaintext
appcircle organization role clear [options]
```

## Options

```plaintext
      --organizationId <uuid>  Organization ID [Optional] (default: "current")
      
      --userId <uuid>          User ID
```
