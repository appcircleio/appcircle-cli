# `appcircle distribution upload`

Upload your mobile app to selected distribution profile.

```plaintext
appcircle distribution upload [options]
```

## Options

```plaintext
     --distProfileId <uuid>  Distribution profile ID
     --message <string>      Release notes
     --app <path>            App path
```
